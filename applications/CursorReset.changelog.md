# История обновлений — CursorReset

[Описание](./CursorReset.md) | **История обновлений**

🚀 Cursor Reset Tool — Обновление v2.6.21+ (VPN Safe Mode)
Это обновление включает в себя критические исправления для обхода новых механизмов отслеживания лимитов в последних версиях Cursor AI (0.45.x и выше).

🛠 Что изменилось в этом обновлении:
🔒 Повышенная анонимность:
Перешли на генерацию 64-символьных Hex ID (Standard VS Code machineId).
Удален префикс authuser_, который служил маркером для серверов Cursor при использовании инструментов сброса.
🧹 Глубокая очистка следов:
Теперь удаляются базы данных storage.vscdb (LevelDB), Local State, а также папки crashpad, logs и cachedData.
🚪 Принудительный выход из аккаунта:
Утилита полностью очищает объект cursorAuth (токены доступа и обновления). Это необходимо, так как старые сессии могут сохранять привязку к лимитам даже после смены ID.
🛡 Защита конфигурации:
Файл storage.json теперь автоматически помечается атрибутом ReadOnly (Только для чтения) после записи новых данных. Это предотвращает попытки приложения перезаписать ваши новые ID старыми значениями.
📝 Важные рекомендации:
Запуск от имени администратора: Это необходимо для корректной работы с реестром и системными файлами.
Новый аккаунт: Старый аккаунт уже помечен лимитом на стороне сервера — использовать его бесполезно.
Смена IP: Перед созданием нового аккаунта рекомендуется сменить IP (использовать VPN или перезагрузить модем).
