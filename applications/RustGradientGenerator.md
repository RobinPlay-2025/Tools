# Rust Gradient Generator 🎨

**Rust Gradient Generator** — это быстрый и удобный инструмент для администраторов серверов Rust, предназначенный для мгновенного создания многоцветных градиентов текста. Экономьте время на настройке конфигов и плагинов, генерируя профессиональный градиентный код в несколько кликов.

## 🌟 Основные возможности

*   **5-точечный градиент**: Поддержка до 5 контрольных точек для создания максимально плавных и реалистичных цветовых переходов.
*   **HSV Color Picker**: Полноценная палитра с выбором оттенка, насыщенности и яркости. Никаких ограничений — выбирайте любой из миллионов цветов.
*   **Живой предпросмотр**: Текст меняет цвет мгновенно при перемещении курсора по палитре. Вы сразу видите финальный результат.
*   **HEX-контроль**: Возможность ручного ввода точных HEX-кодов (#RRGGBB).
*   **Оптимизация для Oxide/Carbon**: Генерация кода в формате `<color=#HEX>T</color>`, который нативно поддерживается игровым чатом и UI плагинов Rust.
*   **Копирование в один клик**: Быстрый экспорт готового кода в буфер обмена с визуальным подтверждением.

## 🚀 Почему это полезно?

Администраторам часто приходится вручную подбирать цвета для приветственных сообщений, объявлений или UI-панелей. Этот инструмент автоматизирует процесс:
1. Введите текст.
2. Настройте до 5 цветов на палитре.
3. Скопируйте готовый код.
**Больше не нужно вручную прописывать каждый символ!**
