# 🚀 Cursor Reset Tool [VPN Safe Mode]

Специализированная утилита для глубокого сброса идентификаторов (HWID) редактора **Cursor AI** с сохранением работоспособности VPN-сервисов.

---

## 🌟 Ключевая особенность: VPN Safe
Многие утилиты сброса меняют системный `MachineGuid`, что приводит к созданию "нового устройства" в ваших VPN-профилях (например, **[VpnSpace](https://t.me/space_connect_bot?start=573F3)**). Это быстро исчерпывает лимит подключенных устройств.

**Cursor Reset Tool [VPN Safe Mode]** работает иначе:
- ✅ Сбрасывает лимиты Cursor через `storage.json`.
- ✅ Очищает все следы предыдущих сессий.
- ❌ **НЕ трогает** системный HWID. Ваш VPN продолжит работать без лишних регистраций.

---

## 🛠 Что делает утилита?

1.  **[1/4] Kill Processes**: Завершает все процессы `Cursor.exe` для доступа к файлам.
2.  **[2/4] Deep Clean**: Полная очистка кеша, Cookies, Local Storage и удаление папки `backups`.
3.  **[3/4] Registry Reset**: Очистка записей в реестре Windows (`Software\Cursor`).
4.  **[4/4] ID Generation**: Генерация новых случайных `machineId`, `macMachineId`, `devDeviceId` и `sqmId`.

## 📖 Инструкция по использованию

### ⚡ Быстрый запуск
1. Скачайте `CursorReset.exe`.
2. Запустите файл (программа автоматически запросит права **Администратора** через UAC).
3. Дождитесь надписи `[ГОТОВО]`.
4. Создайте новый аккаунт в Cursor и работайте!

## ⚠️ Рекомендация
Для максимальной эффективности после сброса рекомендуется **сменить IP** (переподключить VPN) перед входом в новый аккаунт Cursor.
